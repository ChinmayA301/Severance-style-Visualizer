## Overview

This project is an immersive web application designed to replicate the eerie, retro-futuristic corporate aesthetic of Lumon Industries from the show *Severance*. It utilizes React, TypeScript, and Tailwind CSS to construct an unsettlingly precise and performant interactive 3D grid.

### Features
- **3D Isometric Bullpen Grid**: Navigate the Lumon microdata refinement grid with vintage CRT styles and hover effects.
- **Segment Detail Panel**: Dive into specific employee segments with detailed typography and the classic Lumon green interface.
- **Surveillance Hallucination Event**: A randomized sub-system that occasionally triggers a psychological "override" popup.
  - Features real TV show cast diptychs with custom CSS morph sequences.
  - Hard-cuts with deep analog glitches, chromatic aberration, and authentic VHS noise mapping.
  - Subliminal handbook quotes encoded underneath every trigger.
- **Comparison Engine**: Stack two different sectors manually to view cross-department relations.

## Technology Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the frontend server.
4. If you're an authorized MDR executive, you may proceed to your segment. 

### Development Notes

The Surveillance event can be manually triggered via an invisible 16x16px button located at the absolute top-right bounds of the window. Use cautiously.
